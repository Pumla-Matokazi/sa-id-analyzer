South African ID Number Analyzer (CLI Tool)

Description:
The South African ID Number Analyzer is a Python command-line tool that validates and extracts key information from a 13-digit South African ID number. The tool checks for correct formatting and provides users with detailed insights, including date of birth, age, gender, and citizenship status.

Key Features:

Validates ID numbers for correct length and numeric format

Extracts date of birth and calculates age

Determines gender from the sequence number

Identifies citizenship status

Handles invalid input gracefully with clear error messages

Skills Demonstrated:

Python fundamentals: string manipulation, conditionals, functions

Input validation and error handling (try/except)

CLI interface design

Data parsing and real-world problem solving
